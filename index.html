Real-Time Patient Monitoring System

This project demonstrates a real-time system built using an ESP32 and FreeRTOS. It simulates a healthcare device that monitors a patient’s signal and triggers an alarm when unsafe conditions are detected. The system prioritizes timing and safety.

Demo Video

Features

  • Sensor sampling every 100 ms
  • Real-time alarm detection
  • Emergency button using ISR
  • Heartbeat LED
  • Communication task
